1. 如何设置管理员卡片？
在默认设置下，管理员密码为7091。在“欢迎使用”界面，依次输入"8", "NO", "*", "7", "0", "9", "1", "OK"之后，即可进入设置管理员卡片的界面。

2. 如何修改管理员密码？
在已经设置好管理员卡片的前提下，通过刷管理员卡进入“设置”界面，依次选择“管理”、“密码”菜单， 即可进入设置管理员密码的界面。

3. 如果忘记了管理员密码，且管理员卡片也已丢失，有没有方法重新设置管理员卡片? 
嘿，伙计，你也太粗心大意了。还好，我就料想到了你会犯这样的错误。
在“欢迎使用”界面，依次输入"*", "#", "NO", "#", "NO", "*", "OK"之后，即可进入设置管理员卡片的界面。

4. 如何恢复出厂设置，擦除所有用户数据(自定义的管理员密码、管理员卡片、所有箱子的信息)
1）在“欢迎使用”界面，依次输入"8", "NO", "#", "7", "7", "5", "2", "OK"之后，即可恢复出厂设置。在恢复出厂设置的过程中，蜂鸣器会长鸣，整个过程持续约10秒左右。蜂鸣器停止鸣叫后，系统会重新启动以完成恢复出厂设置。
2）不要轻易执行“恢复出厂设置”操作，后果你知道的――所有用户数据丢失，你会被要求卷铺盖走人的。

5. 管理员密码与管理员卡片的关系，好像很复杂的样子？
1）要想设置管理员卡片，必须知道管理员密码（默认密码是7091）。设置方法见1。
2）要想修改管理员密码，必须持有管理员卡片。修改方法见2。
3）方法3属于绝密，千万不能让管理员之外的任何一个人知道。竟然还有这样逆天的邪恶途径，这个方法最好不要让顾客知道，直到他确实遇到了这样的情况，才能将此方法告知于他。切记！
